Description: 室内外机容量组合异常
Solutions:
1.  检查内外机容量配比是否错误
2.  检查接线是否错误
3.  检查容量拨码开关是否错误
4.  检查系统编号设定是否错误
5.  检查是否存在通讯错误、断线或误配线

Code: F1
Description: 正常内机掉线或多管机室外冻结温度传感器TL4异常
Solutions:
1.  情况一：一台室内机出故障，其他室内机在收到这台室内机发出的数据信号之后超过60分钟没有再收到它发出的数据信号，在其他室内机的遥控器上将显示故障代码。
2.  情况二：检查多管机传感器配线是否误接线、断线或短路
3.  检查基板是否故障

Code: F2
Description: 选购件加热器、加湿器等异常或多管机室外气管温度传感器Tg4异常
Solutions:
1.  情况一：检查选购件保护装置是否作动、保护装置是否异常、室内风扇是否停止
2.  情况二：检查传感器配线是否误接线、断线或短路
3.  检查基板是否故障

Code: F3
Description: 外机或者内机编号重复/线控器地址设定错误
Solutions:
1.  情况一：检查系统中是否有重复的内外机地址号系统号。室外机接通电源三分钟后，如果设定的室内机的编号重复，将显示故障代码。
2.  情况二：检查同一水系统内的不同的室内机线控器是否设为同一地址

Code: F4
Description: 室内外机组合错误或品牌错误
Solutions:
1.  没有特别备注

Code: F7
Description: 连接的水模块的编号设置错误
Solutions:
1.  检查室外机机能选择n3功能设定值是否错误
2.  检查水模块电源是否异常

Code: F8
Description: 室外机保护电路动作
Solutions:
1.  检查室外机高压开关是否断开
2.  检查接线是否错误
3.  检查室外机基板的配线是否有错误接线

Code: F9
Description: 定速压机过电流或者零电流
Solutions:
1.  检查是否过电流、零电流
2.  检查电流互感器是否异常
3.  检查电源电压是否异常

Code: FA
Description: 室外机组合模块台数大于四台
Solutions:
1.  检查室外机与子机是否组合错误
2.  检查电源设定是否错误
3.  检查主机和子机的机种或电压设定是否不同

Code: FB
Description: 室外机主机与子机通讯异常/部分从机丢失异常
Solutions:
1.  检查接线是否错误或端子是否松动
2.  检查与子机的通信是否连续30秒不通

Code: FC

Solutions:
